Selamat datang di blog saya!

Perkenalkan, nama saya Madyan dan bisa dipanggil Ian. Saya tinggal bersama orang tua dan adik saya di Kabupaten Kediri, Jawa Timur.

Saya suka dengan teknologi web modern khususnya web berbasis JavaScript. Saat ini saya sedang mendalami pustaka React dan kerangka kerja Next.js.

Meskipun begitu, saya tidak hanya berfokus pada satu bahasa pemrograman saja. Saya pernah berperan sebagai pengembang _full-stack_ menggunakan kerangka kerja Python, yaitu Django dan mengembangkan beberapa modul Odoo, yang sebelumnya dikenal sebagai OpenERP.

Saya cukup antusias untuk memperluas pengetahuan dan kemampuan teknis saya dalam membangun web, khususnya JavaScript. Karena inilah blog ini dibuat.
